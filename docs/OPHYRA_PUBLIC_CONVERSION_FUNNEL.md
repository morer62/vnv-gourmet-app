# Ophyra Public Conversion Funnel

## Purpose

This document tracks the public Ophyra landing pages used for commercial acquisition, conversion and SEO.

The funnel is web-only:

```text
Home -> module or industry landing page -> web signup -> Ophyra workspace
```

The pages must not send commercial users to mobile app signup. Mobile signup remains only for final clients of a specific branded app.

## Implementation

Primary files:

```text
src/Services/OphyraLandingPageService.php
src/views/public/ophyra-landing/index.twig
src/views/public/planner-hub/index.php
src/views/public/planner-hub/index.twig
```

Each page has an `index.php` under its public route folder and uses the shared Twig template.

The home page links to the pages through:

```text
#ophyra-modules
#ophyra-industries
```

## SEO And Schema

Each landing page defines:

* unique title;
* unique meta description;
* canonical URL;
* Open Graph title, description and image;
* Twitter card metadata;
* hero image alt text;
* breadcrumbs;
* internal links to related modules and industries;
* JSON-LD structured data.

Schema currently includes:

```text
WebPage
BreadcrumbList
SoftwareApplication
Service
FAQPage
```

The schema is generated by `OphyraLandingPageService` and printed by `src/views/templates/base.twig` through `schemaJson`.

## Hero Assets

Generated WebP hero assets live in:

```text
public/assets/public/ophyra/modules/
public/assets/public/ophyra/industries/
```

These are launch-ready branded placeholders. They can be replaced later with more polished campaign-specific creative as long as filenames and alt text remain aligned with the landing page topic.

## Conversion Rules

All primary CTAs go to:

```text
/signup
```

Do not route Ophyra commercial acquisition to a mobile signup flow.

Copy should continue to emphasize:

* the hidden cost of scattered operations;
* the practical transformation of moving work into Ophyra;
* the free starter workspace;
* paid modules added only when needed;
* Level 1 support and operational oversight during soft launch.
